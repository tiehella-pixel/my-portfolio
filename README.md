# Personal Portfolio Website

Welcome to my personal portfolio! This project showcases my skills, projects, and experience as a frontend developer, built entirely from scratch using core web technologies.

 Live Demo: https://ella-tieh-portfolio.netlify.app/

# About the Project

This portfolio was designed and developed to reflect my ability to build responsive, modern, and user-friendly web interfaces without relying on frameworks.

It serves as a central hub where recruiters, collaborators, and clients can explore my work, understand my skill set, and easily get in touch.

# Features

 Home Section with background image and personal introduction
 Experience with a download-cv button for access to more information
 Skills Section highlighting my core competencies
 Fully Responsive Design (mobile, tablet, desktop)
 Functional Contact Form with JavaScript handling
 Smooth Scrolling & Interactive UI Elements
 Custom Styling with Pure CSS (no frameworks)

# Built With:
